# 魔方财务对比缺口

更新时间：2026-03-20

## 结论

当前 `idc-finance` 是一套已经可运行的 Go 重建版，但整体完整度仍明显低于魔方财务。

差距不在单个页面，而在三个层面：

1. 领域模块不完整
2. 业务链没有完全闭环
3. 魔方云同步与资源轴没有完整迁入

## 当前已具备

- 平台基础：管理员登录、基础 RBAC、菜单、审计日志
- 客户基础：客户、联系人、实名认证、分组、等级
- 商品基础：产品、价格矩阵、配置项、资源模板
- 主链基础：下单、账单、收款、退款、服务生成
- 门户基础：控制台、订单、账单、服务、工单、钱包、账户
- 报表基础：工作台、报表概览
- 魔方云基础：新工程已具备独立 provider 模块，可做健康检查、实例列表、实例详情、实例动作

## 当前仍是薄页

- 客户详情：缺少 CRM、跟进、附件、通知、销售视角
- 商品详情：缺少升级映射、附加项、自定义字段、上架策略
- 订单详情：缺少审核、取消请求、日志、关联工作区
- 账单详情：缺少流水、余额、信用额、发票/合同、对账
- 服务详情：缺少资源级视图、远端日志、远端同步日志、VNC/资源操作工作台
- 门户：缺少合同、API、下载中心、知识库、完整服务详情工作台

## 缺失的关键模块

### 客户域

- 客户资源池
- 客户跟进/CRM
- 销售归属与业绩
- 推介/分销绑定
- 客户通知与偏好

### 商品域

- 商品组后台深页
- 配置组/附加项
- 自定义字段
- 升降级映射
- 下载中心/资源文件
- 优惠码/代金券

### 订单与财务域

- 购物车
- 续费单
- 升降级单
- 流量包订单
- 取消请求
- 交易流水
- 余额/信用额
- 提现审核
- 发票/合同
- 财务对账
- 支付网关配置

### 服务与资源域

- 统一 Provision/Provider 适配层
- 魔方云远端同步入库
- 磁盘/快照/备份/VPC/安全组/IP
- 状态轮询与同步任务
- 续费规则、到期暂停、到期删除
- VNC/远程控制工作台

### 工单与平台域

- 工单部门
- 状态流转
- 预设回复
- 派单/转派/评价
- 通知模板
- 计划任务/队列
- 插件/HOOK
- OpenAPI
- 上下游资源

## 魔方云迁移状态

### 已带入新工程

- 独立 `provider` 模块
- 魔方云登录鉴权
- 实例列表
- 实例详情
- 常用动作入口

### 还未带入新工程

- 远端实例同步到本地客户/服务/订单
- 门户账号自动补齐
- 资源级同步：IP、磁盘、快照、备份、安全组、VPC
- provider sync log 与任务中心
- 门户服务详情工作台联动
- VNC 页面和控制台链路

## 优先级

1. 先补魔方云同步引擎与资源轴
2. 再补财务闭环：流水、余额、信用额、发票、合同、对账
3. 再补后台详情工作台的深页结构
4. 最后补 OpenAPI、分销、上下游和插件体系
